# Maintenance

## Recommended routine
- update existing notes first
- avoid duplicate summaries
- reindex after meaningful changes
- check for stale entries after deletes/moves
- compress operational memory regularly
- refresh top-level summaries when the base grows

## Why this matters
A memory system usually degrades through:
- duplication
- stale indexes
- overgrown local memory
- summary drift
not through lack of raw text.
