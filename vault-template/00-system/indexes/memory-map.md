# Memory Map (Template)

## Canonical store
This vault is the canonical source of truth for durable user knowledge.

## Sections
- `00-system/` — rules and runbooks
- `10-identity/` — stable identity notes
- `20-preferences/` — preferences and operating style
- `30-projects/` — active projects
- `40-people/` — important people
- `50-decisions/` — major decisions and reasoning
- `60-context/` — current life context
- `70-daily/` — raw daily captures
- `80-summaries/` — top-level summaries
- `90-archive/` — superseded notes

## Rule
Keep durable facts here, not only in temporary local memory.
